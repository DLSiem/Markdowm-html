## Text Formatting

This is **bold** text.  
This is _italic_ text.  
This is ~~strikethrough~~ text.  
This is **_bold and italic_** text.

You can also combine styles:

- **Bold** and _Italic_ in the same line.
- ~~Strikethrough~~ combined with **bold**.
